# helix-git

Upstream: https://github.com/helix-editor/helix

Created on May 18, 2026

## Information

This ebuild is the first in this repository. It was created due to
2 main reasons.

1. The official Gentoo's ebuild for **Helix** only supports `amd64` architecture, which doesn't work for any other architectures (such as `arm64`)
2. Helix has been only getting updates in the upstream repository, it's last current release (as of writing) was released in 2025.
